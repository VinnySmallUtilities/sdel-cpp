Дано
std::vector<std::string>   temp_pattern;

temp_pattern заполнен хотя бы одной строкой.
Строки представляют собой шестнадцатиричные числа (в строковом формате), разделённые пробелами и, возможно, запятыми (либо не разделённые ничем). Например, "00", "00AA00FF", "00AA 00FF", "55AA", "55 AA", "00 AA, 00 FF", "00 AA 00 FF  DD BB". Числа могут быть любого размера.

Необходимо сгенерировать функцию, которая будет преобразовывать эти строки в список массивов byte, которые будут представлять каждую из строк. Сохранить порядок.
То есть нужно:
1. Каждую строку преобразовать в массив однобайтовых значений.
2. Каждый массив уже добавить в итоговый массив, сохраняя порядок, такой же как в исходных строках.

// Тип для массива байтов
using ByteArray = std::vector<uint8_t>;
// Тип для списка массивов байтов
using ByteArrayList = std::vector<ByteArray>;
